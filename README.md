
# 🎫 Sistema Cupom Local - São José do Rio Preto

O **Sistema Cupom Local** é uma plataforma desenvolvida para fortalecer o comércio de São José do Rio Preto/SP, permitindo que lojistas criem, gerenciem e validem cupons de desconto de forma digital, segura e sem intermediários caros.

## 🚀 Funcionalidades Principais
- **Painel do Lojista:** Criação de cupons personalizados.
- **Validação via QR Code:** Segurança total contra uso duplicado de cupons.
- **Analytics Local:** Relatórios de resgates e perfil de consumo por bairro.
- **Interface Responsiva:** Otimizado para celulares (cliente e lojista).

## 🎯 Por que usar o Sistema Cupom Local?
Diferente de grandes plataformas de delivery, nosso foco é o **comércio de bairro**. O lojista mantém 100% do valor da sua venda e paga apenas uma taxa fixa simbólica para manutenção da infraestrutura tecnológica.
